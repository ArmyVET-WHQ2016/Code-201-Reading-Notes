## HTML Â lists allow web developers to group a set of related items in lists.

**Ordered HTML List**
*the list item in the list is a number*

1. <ol>
2.  <li>Coffee</li>
3.  <li>Tea</li>
4.  <li>Milk</li>
5. </ol>


**Unordered HTML List**
*the list begins with a bullet point* 

+ <ul>
+   <li>Coffee</li>
+   <li>Tea</li>
+   <li>Milk</li>
+ </ul>




**HTML Description Lists**
*are made up of set of terms along with the definitions for each of those terms*


<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  <dt>Milk</dt>
  <dd>- white cold drink</dd>
</dl>



**Nested list**
*a second list inside an <li> element to create a nested list or sublist*

- <ul>
-  <li>Coffee</li>
-  <li>Tea
-    <ul>
-      <li>Black tea</li>
-      <li>Green tea</li>
-    </ul>
-  </li>
  <li>Milk</li>
</ul>
